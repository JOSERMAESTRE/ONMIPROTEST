IMFORMACION IMPORTANTE
Dentro de este repositorio encontraras
1. archivo .feature el cual tiene los casos de prueba en formato Gherkin
2. archivo .pdf que contiene reporte de cobertura de las pruebas
3. en la carpeta test-output encontraras el archivo emailable-report que contiene el reporte TESTNG de las pruebas ejecutadas

RECOMENDACIONES
para ejecutar los script deberás instalar Eclipse IDE for Java Developers - 2024-09
en la clase RegisterPageTest en el TestRegister se le sugiere cambiar el correo Masterjr@gmail.com por otro correo para el éxito de la prueba
para ejecutar las pruebas deberás hacerlo desde el archivo testng.xml
las pruebas son ejecutadas en el navegador Chrome Versión 131.0.6778.86 (Build oficial) (64 bits)
